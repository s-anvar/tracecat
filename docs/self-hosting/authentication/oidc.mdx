## Configuration

Set `TRACECAT__AUTH_TYPES` to include `oidc`.

```bash
TRACECAT__AUTH_TYPES=oidc
```

## Instructions

<Steps>
  <Step title="Create an OIDC app in your provider">
    Register a new application and obtain the client ID and secret.
    Set the redirect URL to `<your-domain>/auth/oidc/callback`.
  </Step>
  <Step title="Configure environment variables in Tracecat">
    Set the following in your `.env` file:
    - `OIDC_CLIENT_ID`
    - `OIDC_CLIENT_SECRET`
    - `OIDC_DISCOVERY_URL`
    - `OIDC_GROUP_ROLE_MAP`
    - `OIDC_DEFAULT_ROLE`
    - `OIDC_STRICT_MODE`
  </Step>
  <Step title="Restart Tracecat">
    Run `docker compose up`.
  </Step>
  <Step title="Map provider groups to Tracecat roles">
    Example mapping:
    ```bash
    OIDC_GROUP_ROLE_MAP='{"admins":"ADMIN","users":"BASIC"}'
    OIDC_DEFAULT_ROLE=BASIC
    OIDC_STRICT_MODE=true
    ```
    Group names are compared case-insensitively and any surrounding whitespace is ignored.
  </Step>
</Steps>
